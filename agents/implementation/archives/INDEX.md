# Implementation Archives Index

This index helps navigate historical implementation plans and notes.

## Layout
- `agents/implementation/archives/plan_01.md` ... `agents/implementation/archives/plan_101.md` - early plan history.
- `agents/implementation/archives/plan_102.md` ... `agents/implementation/archives/plan_119.md` - mid-series execution history.
- `agents/implementation/archives/plan_128.md` and `agents/implementation/archives/plan_128_phase_i_checklist.md` - feed/message migration context.
- `agents/implementation/archives/plan_114A.md` ... `agents/implementation/archives/plan_114F.md` - split sub-phases for plan 114.
- `agents/implementation/archives/notes/*` - supplemental notes linked from plans.

## Usage
- Prefer active plans in `agents/implementation/` for current execution.
- Use archives for historical decisions, migration rationale, and old test references.
- When searching history, use `rg` first (for plan number, feature keyword, or route name).
